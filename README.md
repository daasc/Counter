# Counter

Counter is a project created using the TDD method.

## Installation

```bash

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate

# run test
$ npm run test:watch
$ npm run test:coverage
```
